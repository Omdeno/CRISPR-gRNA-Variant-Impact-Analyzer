# CRISPR-gRNA-Variant-Impact-Analyzer

A comprehensive toolkit for assessing CRISPR gRNA functionality considering genetic variants (SNPs, InDels, SVs).

## Components

### 1. gRNA Variant Impact Analyzer
- Parallel processing of gRNA functionality predictions
- Integration of SNP and structural variant data
- Functional classification (Fully_Functional, Reduced_Efficiency, Critical_Failure)

### 2. gRNA Statistics & Visualization
- Publication-ready plots using ggplot2/ggpubr
- Off- and on-target threshold sensitivity analysis
- Homoeologous pairs analysis
